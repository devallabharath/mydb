## MyDB with Electron JS
---

### To start app

- Go to the project root directory in the terminal

- Install the dependicies
    ```sh
    > npm install
    ```
- Finally run the code with Electron
  ```sh
  > electron .
  # or with npm scripts
  > npm start
  ```
